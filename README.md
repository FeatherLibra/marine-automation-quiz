# 轮机自动化刷题软件

一个为轮机自动化复习整理的离线刷题网页应用。项目把题库整理成可直接使用的手机/电脑网页，支持背诵、随机模拟考试、自动判分和历史记录保存。

## 功能

- 1295 道轮机自动化选择题题库
- 随机抽取 50 题模拟考试
- 自动判分、显示正确率和错题明细
- 历史考试记录保存在浏览器本地
- 背诵模式支持按 Day 计划、章节、关键词筛选
- 支持错题、标星、会了等本地记忆标记
- PWA 离线缓存，可添加到手机主屏幕

## 页面

- `index.html`：50 题模拟考试
- `exam.html`：50 题模拟考试备用入口
- `list.html`：背诵题库和筛选复习

## 本地运行

直接打开 `index.html` 可以使用大部分功能。若要测试离线缓存和 PWA，建议启动一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 数据说明

题目数据已经内嵌在静态 HTML 文件中，不依赖后端服务。学习记录使用浏览器 `localStorage` 保存，只存在于当前浏览器。

## 项目定位

这是一个面向个人学习场景的小工具：把零散题库整理成可以反复练习、模拟考试和复盘错题的网页应用。
